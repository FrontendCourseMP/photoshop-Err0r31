# Демченко Егор Максимович 231-322 

Веб-приложение для загрузки, отображения и экспорта изображений с поддержкой форматов PNG, JPG и кастомного формата GB7.

## [Открыть приложение](https://photoshop-demchenko-egor.onrender.com/)

## Запуск проекта

1. Установить зависимости:

```bash
npm install
```
2. Запустить проект в режиме разработки:

```bash
npm run dev
```
